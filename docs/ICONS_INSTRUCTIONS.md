# تعليمات إضافة أيقونات المود

## الملفات المطلوبة

يجب إضافة الملفات التالية في مجلد `/public`:

1. **nightMode.png** - أيقونة القمر (الوضع الليلي)
2. **dayMode.png** - أيقونة الشمس (الوضع النهاري)

## المواصفات الموصى بها

- **الحجم**: 72x72 بكسل أو أكبر (سيتم تغيير الحجم تلقائياً)
- **التنسيق**: PNG مع خلفية شفافة
- **الجودة**: عالية الدقة للحصول على أفضل مظهر

## الموقع

```
my-nextjs-app/
└── public/
    ├── nightMode.png  ← ضع الملف هنا
    └── dayMode.png    ← ضع الملف هنا
```

## ملاحظات

- الأيقونات ستظهر داخل Toggle Switch الاحترافي
- حجم العرض المحدد في CSS: 72px
- الارتفاع: تلقائي (auto) للحفاظ على النسبة
